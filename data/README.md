# data/ — captured logs

Raw captures pulled off the drone for analysis: sensor baselines, motor disturbance maps, etc.

Naming: `<area>/<what>_<conditions>.csv` — e.g. `sensors/idle_flat_30s.csv`,
`motors/spin_motor1_lowspeed.csv`. Keep a one-line note of capture conditions at the top of
each file (power source, drone orientation, what was done) so logs are interpretable later.
